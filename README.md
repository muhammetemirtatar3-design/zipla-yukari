# Zıpla Yukarı

Tarayıcıda oynanan basit 2D zıplama oyunu. Top sürekli zıplar; siz sadece sağa-sola hareket edersiniz.

## Oynatma

`index.html` dosyasını çift tıklayın veya tarayıcıya sürükleyin. Sunucu gerekmez.

## Kontroller

| Girdi | Eylem |
|-------|--------|
| ← → veya A D | Sağa / sola |
| Mobil | Ekranın sol / sağ yarısına dokun |
| Boşluk | Game over sonrası hemen yeniden başla |

## Basamaklar

- **Yeşil** — Kalıcı, sınırsız zıplama
- **Turuncu (!)** — Tek kullanımlık, basınca kırılır

## Teknoloji

HTML5 Canvas, vanilla JavaScript — harici bağımlılık yok.

## Geliştirme

Tüm oyun mantığı `index.html` içinde. Değişiklikten sonra sayfayı yenileyin.
